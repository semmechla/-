Идея возникла из игры "Говорящий бен" с телефоном, здесь всё по аналогии. Бот был написан для обучения работы с апи изначально. Поэтому реализация рандома здесь не через библиотеку random а через yesno.wtf/api. Кому надо могут переделать под локальный рандом ибо так быстрее.
